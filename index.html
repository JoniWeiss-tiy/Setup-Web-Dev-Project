Set up Development Environment

Webpack:

  1. Create directory structure
    • project directory
    • --public
    • ----bundle.js
    • --src
    • ----scripts
    • ------entry.js
    • ----styles
    • ------main.sass
  2. $ npm init
  3. $ npm install webpack --save-dev
  4. Create webpack.config.js file and populate it.
  5. Add entry.js file and add requires to it.
  6. Install style, sass, and css loaders: $ npm install css-loader style-loader npm-sass sass-loader --save-dev
  7. Run webpack: $ webpack --watch
  8. Install webpack web server: $ npm install webpack-dev-server --save-dev
  9. Open a new terminal window and run the webpack server: $ node_modules/.bin/webpack-dev-server
  10. NOTE: Both terminals windows need to be open and running webpack --watch and webpack-dev-server.
  11. Navigate to localhost:8080/webpack-dev-server/

Mocha:

  1. Install mocha: $ npm install mocha --save-dev
  2. Create tests directory (under src), and tests.js file for the tests.
  3. Add tests to the tests.js file.
  4. Run tests like this: $ mocha ./src/tests
  5. Edit webpack.config.js file to add the command line for mocha tests under the scripts entry.
